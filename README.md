# 掲示板システム「Xちゃんねる」について

本プロジェクトをご覧いただき、誠にありがとうございます。<br>
本システムは、個人で初めて開発した掲示板アプリケーション「Xちゃんねる」です。<br>
X（旧Twitter）と2ちゃんねるの要素を組み合わせたシステムとなっております。

### (3) 主な機能
 * 会員管理機能<br>
 データベースに登録された会員情報のみログイン可能。

 * 投稿管理機能<br>
 掲示板に投稿された内容は、投稿者本人のみ削除可能。

 * 検索機能<br>
 投稿者名・投稿内容を検索し、一致する投稿を一覧表示。

## 4. 今後の課題および追加予定の機能
 * 投稿の編集機能<br>
   投稿一覧に表示される削除機能に加え、編集機能を実装予定。編集後も投稿日時を変更せずに表示できる仕組みを導入予定。

## 5. システムの実行手順
### (1) H2 Databaseのインストール
1. <a href="https://www.h2database.com/html/main.html" target="_blank">公式サイト</a>
にアクセスし、「Download」内の「All Platforms」のリンクをクリックしてZIPファイルをダウンロード。<br>
2. ダウンロードしたZIPファイルを解凍。<br>
3. ターミナルを起動し、以下の手順でH2 Databaseを起動。<br>
cd <H2 Databaseのディレクトリ>/bin
./h2.sh


### (2) H2 Databaseへの接続
1. H2コンソールにアクセスし、保存済み設定から「Generic H2(Server)」を選択。<br>
2. JDBC URLに jdbc:h2:tcp://localhost/~/dokoTsubu を指定。<br>
3. 「接続」ボタンをクリックし、データベースに接続。<br>

### (3) アプリケーションの起動
1. index.jsp を実行。
